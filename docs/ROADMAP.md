# Photo Overlay Studio - Roadmap

## Phase 1 - Quick Wins
- [ ] **Wasserzeichen / Custom Text** - Eigenen Text (z.B. Instagram-Handle, Copyright) als zusaetzliche Zeile im Overlay
- [ ] **Datum & Uhrzeit** - EXIF DateTimeOriginal als optionales Overlay-Feld
- [ ] **Transparenter PNG-Layer Export** - Nur das Overlay als transparente PNG exportieren (ohne Foto), zur Weiterverarbeitung in anderen Apps

## Phase 2 - Mehr Kontrolle
- [ ] **Farbanpassung** - Akzentfarbe des Overlays frei waehlbar (Color Picker)
- [ ] **Overlay-Groesse** - Schriftgroesse per Slider steuern
- [ ] **Batch-Verarbeitung** - Mehrere Fotos gleichzeitig mit demselben Overlay-Stil exportieren
- [ ] **Live Drag & Drop** - Overlay-Position direkt im Preview per Maus/Touch verschieben

## Phase 3 - Erweiterte Features
- [ ] **Mehr Kamera-Marken** - Canon, Nikon, Fujifilm, Panasonic Modell-Mapping (wie Sony bereits vorhanden)
- [ ] **Preset-Sharing** - Overlay-Einstellungen per URL teilen
- [ ] **PWA / Offline** - Als App installierbar mit Service Worker
- [ ] **Social Media Formate** - Automatisches Zuschneiden fuer Instagram Story (9:16), Post (1:1, 4:5), etc.

## Phase 4 - Pro Features
- [ ] **RAW-Support** - DNG/ARW Vorschau mit EXIF-Extraktion
- [ ] **GPS-Karte** - Kleine Kartenvorschau wenn Standortdaten in EXIF vorhanden

## Phase 5 - Native App
- [ ] **iOS App** - Native iOS App mit Share Extension (Foto direkt aus der Galerie teilen)
